### 💰 T-Cash
**Сервис управляемого кэшбэка**

[![T-Cash](https://img.shields.io/badge/Project-T--Cash-green?style=for-the-badge)](https://github.com/Vova1707/TCash)

Backend-сервис для решения задачи управляемого кэшбэка:
- ⚙️ Админка настраивает категории, правила отбора и бюджеты
- 📋 Клиент получает персональный список категорий (offers/run)
- ✅ Сервер гарантирует лимиты, идемпотентность и проверку бюджетов

**Технологии:** `Java` `Spring Boot` `PostgreSQL` `SQL` `REST API`
